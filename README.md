# KatNews

A front end application application displaying aggregated news articles.

## Links
 - A live version of the site can be found at https://katnews.netlify.com/
 - The repository can be at  https://github.com/ammeLacy/kat_news
 - The back end is hosted at https://kat-news.herokuapp.com/api
 - The back end repository can be found at https://github.com/ammeLacy/nc-news

## Installation 
#### Git
 - On GitHub if you require your own repository set that up before cloning the repository
 - Clone the repo from https://github.com/ammeLacy/kat_news and then (if requiring your own repository *git remote remove origin* then **<YOUR-GITHUB_URL>**) git remote add origin 
  
  #### Dependencies
 - The project has the following dependencies
      - @fortawesome/free-solid-svg-icons 5.10.2
	 - @fortawesome/fontawesome-svg-core 1.2.22
	 - @fortawesome/react-fontawesome 0.1.4
	 - @reach/router 1.2.1
	 - query-string 6.8.2
	 - axios 0.19.0
	 - bootstrap 4.3.1
	 - react 16.9.0
	 - react-dom 16.9.0
	 - reactstrap 8.0.1

 -  To install the project use *npm install*
#### Scripts 

 -  *npm start* runs the app in the development mode on local host 3000, if this not opened automatically enter  http://localhost:3000
 The page will reload if you make edits. You will also see any lint errors in the console.
